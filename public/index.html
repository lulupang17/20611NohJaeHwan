독거노인들을 위한 서비스 모음ZIP

독거노인을 위한 응급 상황 대비 웹 서비스 안내

사용 방법

회원가입

회원가입 시 사용자의 이메일과 보호자 이메일을 입력해야 하며, 반드시 실제 사용 중인 이메일을 입력해야 합니다.

  • 사용자 이메일: 웹사이트의 각종 알림을 수신합니다.
  • 보호자 이메일: 응급 상황 발생 시 알림이 전송됩니다.

로그인 후 숙지 사항

  1. 출석체크 기능
  2. 생활 팁 제공 페이지
  3. 긴급 전화번호 페이지

출석체크 페이지

출석체크 페이지에 접속하면 출석체크 버튼이 표시됩니다.

  • 버튼을 클릭하면 출석 기록이 저장됩니다.
  • 주의: 3일 이상 출석체크를 하지 않을 경우, 시스템은 응급 상황 가능성을 감지하고 보호자에게 자동으로 긴급 알림을 발송합니다.
  • 기능 테스트: 알림메시지 테스트하기 버튼을 눌러 알림이 정상적으로 작동하는지 확인할 수 있습니다.

회원정보 수정 페이지

회원 정보 변경 또는 알림 수신 설정을 수정하려면 회원정보 수정 페이지를 활용하세요.

  • 회원정보 수정 버튼: 이름, 이메일 등 회원 정보를 업데이트합니다.
  • 알림 수신 버튼: 보호자에게 발송되는 긴급 알림 수신 여부를 켜거나 끕니다.
  • 로그아웃 버튼: 로그인 세션을 종료합니다.
